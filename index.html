Appikon

Web-based iOS app icon preview solution

Download .zip Download .tar.gz View on GitHub

What is APPIKON

A quick and elegant way let visual designers have ability to review App icons on device.

HOW TO USE

Creating each size of app icon, upload to public space and review.

Design

  1. Create you icon
  2. Export all sizes (57x57, 114x114, 72x72, 144x144 ) of icon image in PNG format

Upload

Upload all files to a public space. (e.g. Dropbox public folder)

Review

Open the URL on Mobile Safari then "Add to Home Screen"
